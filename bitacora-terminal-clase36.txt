1. ¿Qué es un sistema operativo?
Respuesta:
Es el software principal de un computador que administra los recursos del equipo y 
permite ejecutar programas y aplicaciones.

2. ¿Qué es el código abierto?
Respuesta:
Es un tipo de software cuyo código fuente está disponible para que las personas 
puedan estudiarlo, modificarlo y mejorarlo.

3. ¿Qué significa soberanía tecnológica?
Respuesta:
Es la capacidad de una comunidad o país de controlar, comprender y decidir sobre las
tecnologías que utiliza sin depender completamente de otros.

4. ¿Qué es Linux?
Respuesta:
Es un sistema operativo de código abierto basado en Unix, utilizado en computadores, 
servidores y otros dispositivos.

5. ¿Qué es Debian?
Respuesta:
Es una distribución de Linux conocida por su estabilidad, seguridad y uso de software libre

6. ¿Qué es un servidor?
Respuesta:
Es un equipo o sistema que ofrece servicios, datos o recursos a otros dispositivos 
conectados a una red.

7. ¿Qué es una terminal?
Respuesta:
Es una herramienta que permite comunicarse con el sistema mediante comandos de 
texto para realizar tareas

9. ¿Qué comando me pareció más útil y por qué?
Respuesta:
El comando ls me pareció muy útil porque permite conocer rápidamente qué archivos 
y carpetas existen en una ubicación y facilita la organización del trabajo.

10. ¿Qué aprendí sobre el uso responsable de la terminal?
Respuesta:
Aprendí que la terminal es una herramienta poderosa que debe utilizarse con cuidado, 
verificando los comandos antes de ejecutarlos para evitar errores o pérdida de información.

11. ¿Qué información no debo publicar en capturas o evidencias?
Respuesta:
No debo publicar contraseñas, claves privadas, tokens, direcciones IP sensibles, información 
personal o datos de acceso a servidores.

12. ¿Qué cuidados debo tener al trabajar en un servidor?
Respuesta:
Debo usar comandos con precaución, proteger las credenciales, realizar copias de seguridad y 
evitar modificar archivos importantes sin verificar.

13. ¿Cómo se relaciona la terminal con la soberanía tecnológica?
Respuesta:
La terminal permite conocer y controlar el funcionamiento del sistema, especialmente usando 
herramientas de código abierto como Linux, lo que ayuda a tener mayor autonomía sobre la 
tecnología utilizada.
