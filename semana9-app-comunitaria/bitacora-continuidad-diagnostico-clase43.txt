1. ¿Qué es continuidad operativa?
Respuesta:
Es la capacidad de una aplicación o servicio para mantenerse funcionando de forma 
estable y disponible, incluso cuando ocurren errores, fallos o interrupciones.

2. ¿Qué es PM2?
Respuesta:
PM2 es un administrador de procesos para aplicaciones Node.js que permite iniciar, 
supervisar, reiniciar y mantener una aplicación ejecutándose en segundo plano.

3. ¿Qué comando usé para iniciar la aplicación con PM2?
Respuesta:
npx pm2 start server.js --name app-semana9

4. ¿Qué comando usé para listar procesos?
Respuesta:
npx pm2 list

5. ¿Qué comando usé para detener la aplicación?
Respuesta:
npx pm2 stop app-semana9

6. ¿Qué comando usé para reiniciar la aplicación?
Respuesta:
npx pm2 restart app-semana9

7. ¿Qué comando usé para revisar logs?
Respuesta:
npx pm2 logs app-semana9 --lines 20

8. ¿Qué error simulé?
Respuesta:
Simulé un error en la aplicación, como una configuración incorrecta,
 una variable faltante o un fallo que provocara una respuesta inesperada del servidor.
